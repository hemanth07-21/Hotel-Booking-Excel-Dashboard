# Hotel-Booking-Excel-Dashboard

📊 Hotel Booking Analysis Dashboard (Excel)
🔹 Project Overview
This project presents an interactive Hotel Booking Analysis Dashboard built using Microsoft Excel.
The dashboard analyzes hotel booking data to identify trends, cancellations, and performance comparison between City Hotel and Resort Hotel.
The goal of this project is to demonstrate strong skills in:
Data Cleaning
Data Analysis
Data Visualization
Dashboard Design
🔹 Tools & Techniques Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Donut Charts
Bar Charts
KPI Cards
Conditional Formatting
🔹 Dataset Description
The dataset contains hotel booking records including:
Hotel Type (City / Resort)
Arrival Date (Year)
Booking Status
Cancellation Data
Customer Information
Data was analyzed for the period: 📅 2015 – 2017
🔹 Key Performance Indicators (KPIs)
📌 Total Bookings: 119,390
📌 Total Cancellations: 44,224
📌 Year-wise Booking Trends
📌 City Hotel vs Resort Hotel Comparison
🔹 Dashboard Features
✅ Interactive Year Filter using Slicer
✅ Dynamic Charts updating based on selection
✅ Donut Chart for Booking Distribution
✅ Bar Chart for Year-wise Analysis
✅ Clean and professional layout design
