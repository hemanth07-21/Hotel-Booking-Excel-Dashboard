📊 Hotel Booking Analysis Dashboard (Excel)

🔹 Project Overview

This project presents an interactive Hotel Booking Analysis Dashboard built using Microsoft Excel.

The dashboard analyzes hotel booking data to identify:

Booking trends over time

Cancellation patterns

Performance comparison between City Hotel and Resort Hotel

The main objective of this project is to demonstrate strong skills in:

✅ Data Cleaning

✅ Data Analysis

✅ Data Visualization

✅ Dashboard Design

🛠 Tools & Techniques Used

Microsoft Excel

Pivot Tables

Pivot Charts

Slicers

Donut Charts

Bar Charts

KPI Cards

Conditional Formatting

📂 Dataset Description

The dataset contains hotel booking records from 2015 – 2017.

Key Data Fields:

🏨 Hotel Type (City / Resort)

📅 Arrival Date (Year)

📌 Booking Status

❌ Cancellation Data

👥 Customer Information

📈 Key Performance Indicators (KPIs)
Metric	Value
📌 Total Bookings	119,390
❌ Total Cancellations	44,224
📊 Year-wise Booking Trends	2015 – 2017
🏨 Hotel Comparison	City vs Resort
📊 Dashboard Features

✅ Interactive Year Filter using Slicer

✅ Dynamic charts updating based on selection

✅ Donut Chart for Booking Distribution

✅ Bar Chart for Year-wise Analysis

✅ Clean and professional layout design
